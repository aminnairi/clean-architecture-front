# clean-architecture-front

Exploration for using a Clean Architecture for both React & Vue front-ends.

## Requirements

- Node
- NPM

## Installation

```bash
npm install
```

## React

### Watch

```bash
npm run watch
```

### Build

```bash
npm run build
```

### Start

```bash
npm start
```

## Vue

### Watch

```bash
npm run watch
```

### Build

```bash
npm run build
```

### Start

```bash
npm start
```
